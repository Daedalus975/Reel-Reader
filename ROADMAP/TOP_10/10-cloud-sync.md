# 10 — Cloud Sync (playback positions, optional)

**Summary:** Optional cloud sync for playback positions, playlists, and device state (opt-in).

**Acceptance criteria:**
- Users can opt-in to cloud sync and see devices/playlists synced across devices.
- Conflicts handled via last-write-wins or manual resolution UI.

**Feature flag:** `feature_cloud_sync`
**Estimate:** 6–10 days
