# 03 — Offline Downloads Manager

**Summary:** Implement a download manager for offline playback (queue, retries, storage quota, scheduled downloads).

**Acceptance criteria:**
- Users can queue media for download and see progress.
- Downloads persist through restarts and support retry/backoff.
- Storage usage is visible and managed.

**Feature flag:** `feature_offline_downloads`
**Estimate:** 6–10 days
