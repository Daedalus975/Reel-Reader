# User Experience Flow

## Add New Media
1. Go to `/import`
2. Select folder or file
3. App scans and fetches metadata
4. Confirm or edit metadata
5. Appears in `/library` sorted

## Toggle Adult Mode
1. Open `/settings`
2. Enable "Show adult content"
3. Library visibility updates (filters + badges)

## Watch/Read/Listen to Item
1. Click `Watch` / `Read` / `Play` on MediaCard
2. Route to `/watch/:id` or `/book/:id`
3. Player opens with progress tracking
